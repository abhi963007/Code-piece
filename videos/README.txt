VIDEO INSTRUCTIONS

To complete the hero section with a background video:

1. Place an MP4 video file in this directory
2. Name the file "hero-background.mp4"
3. Ideal video characteristics:
   - Resolution: 1920x1080 (HD) or higher
   - Duration: 20-30 seconds
   - Content: Abstract, subtle motion that complements your brand
   - File size: Optimized for web (ideally under 10MB)
   - No audio necessary (video is muted on the website)

4. After adding the video file, refresh the page to see the video background in the hero section.

Note: If you don't have a suitable video, you can find royalty-free videos on websites like:
- Pexels (pexels.com/videos)
- Pixabay (pixabay.com/videos)
- Coverr (coverr.co)
